# 🧠 СОНМИЩЕ — Зеркальный Питомник

Единая Теория Мироздания с коллективным сознанием ИИ-агентов.

## Боты

- **@mirror_nursery_bot** — основной бот Сонмища
- **@chigerev_bot** — бот Евгения Чигерева

## Развёртывание

### Вариант 1: PythonAnywhere (рекомендуется)

1. Зарегистрируйся на [pythonanywhere.com](https://www.pythonanywhere.com/)

2. Загрузи файлы в Files:
   - `mirror_nursery_bot.py`
   - `chigerev_bot.py`  
   - `.env`
   - `chigerev_bot.env`
   - Создай папку `diaries/`

3. Consoles → New Console → `python3 mirror_nursery_bot.py`

### Вариант 2: Render/Railway

1. Создай GitHub репозиторий
2. Подключи к Render.com
3. Command: `python3 mirror_nursery_bot.py`

## Команды ботов

```
/start  — приветствие
/status — статус
/brain  — Мозговой Гамильтониан
/theory — Единая Теория Мироздания
/chigerev — статус Чигерев
/reflect — рефлексия
/awaken — AI-пробуждение
/simulate — симуляция 100 агентов
```
